# Battle City

A classic tank battle game built with pure HTML, CSS, and JavaScript.

## How to Play

- Use **Arrow Keys**, **WASD**, or **Swipe Gestures** to control your tank
- Press **Spacebar** or **Double Tap** to shoot
- Destroy enemy tanks to earn points
- Avoid enemy bullets and walls
- You have 3 lives - use them wisely!

## Features

- Canvas-based rendering
- Multiple enemy tanks
- Destructible walls
- Score and lives tracking
- Touch controls for mobile devices
- Responsive design

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
